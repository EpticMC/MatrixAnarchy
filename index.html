About:

MatrixAnarchy is an anarchy server that started on 4th of November, 2020. The goal of the server is to be a lag-free and queue-free anarchy server with permanent dupes that's always on the latest version and supports older/newer versions. Currently you can join using any version from 1.12 to 1.16. The server is running on 1.16.4.
